init.sql - Contains SQL commands to
1) Recreate the database 'eclt5830' 
   Note: database name is harcoded in "/config/mysql.js" and also in "init.sql"
2) Recreate three tables: users, items, redeemed_items
3) Populate the tables with some test data (the test data is hardcoded in "init.sql")

init.js - Execute the SQL commands in init.sql 
(Can be executed as "node init.js")
